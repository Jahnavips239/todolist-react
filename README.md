# 📝 Todo App

A responsive Todo List application built using React.

## 🚀 Features
- Add tasks
- Delete tasks
- Enter key support
- Clean and modern UI

## 🛠️ Tech Stack
- React.js
- CSS

## 📦 Installation

```bash
npm install
npm start
```
